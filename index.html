Wrap-up: you can use nested css selectors untill you sure:
1) there will be no nested blocks which use the same classes (or no nested blocks at all)
2) there will be no wrapping blocks using the same classes
3) there will be no global css classes with the same name
4) as BEM utilize minimum priority selectors - it is hard to use BEM for some block in a project 
   already polluted with global styles which use tag names. But otherwise - bem class names look pretty
   unique and thus should not be affected by the styles assigned to global, unprefixed classes.
